About Vector



Vector is a modern and simplistic approach to the playing and management of scavenger hunt games for a wide array of people.


Audience


Vector will be targeted at organizations or groups who host outdoor or party activities for leisure or fundraising purposes.

Purpose


To increase the ease in organizing scavenger hunt games by providing a platform that will allow for users to quickly create and participate in large scale scavenger hunt games, without the need to physically place objects at target positions.

How it works


A user can either be a player or the creator of a game. The creator is able to set the details of the game including: the name, start and end times, the location and details of clues. The player is able to search and play games. After a game has ended, the winner of the game will be determined as either the player in the game with the highest score or the first player to achieve the maximum score.
